# Oman Airports Detail

An interactive geographic visualization providing location data and details about various airports situated in Oman.

## Overview
This repository contains an interactive HTML map focusing on Oman's airports, allowing users to explore different aviation hubs across the country.

## Features
- **Airport Locations**: Precise geographic placement of major and minor airports in Oman.
- **Interactive Experience**: Built for researchers, geographers, or aviation enthusiasts interested in Oman's infrastructure.
